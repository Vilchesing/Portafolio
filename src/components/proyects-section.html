Whatsapp

Implementé una aplicación web de mensajería en tiempo real utilizando Service Workers. Esta tecnología me permitió almacenar mensajes localmente y sincronizarlos cuando el dispositivo se vuelve a conectar, brindando una experiencia de usuario similar a una aplicación nativa. Además, exploré características como notificaciones push y actualizaciones en segundo plano para enriquecer aún más la experiencia del usuario
